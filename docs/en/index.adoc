== Build Instructions

You can build using Docker:

[source,bash]
----
docker build -t adoc-builder .
docker run --rm -v $(pwd)/docs:/docs:ro -v $(pwd)/output:/output adoc-builder
----

Build artifacts will be output to the `output/` directory.

== Summary

By using this template, you can easily convert AsciiDoc documents to HTML and PDF.
